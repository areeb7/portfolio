Predictor

This Project is used to predict the salary a client based on his/her personal and professional factors like country of origin, sex , family, profession, education, professional experience, and many other factors. Machine learning revolves around the data i.e why a huge amount of user data was required to build the model. Python is best used for data visualization, as it has enormous packages, most commonly used packages in this project are NumPy , pandas, Scikitlearn, Matplotlib etc. This project uses supervised machine learning, and the algorithm used to achieve the highest test score was decision Tree classifier. The highest score was 85% i.e the project predicts the salary of the clients correctly 85 times out of 100.
